# EchoSteps

EchoSteps is a responsive React web application that helps users explore walking routes, record walks, discover nearby places, and save memories from their journeys.

## Features

- Responsive modern user interface
- Home page with featured walking routes
- Discover page with route cards and images
- Walk recording screen with start/stop timer
- Distance and step tracking preview
- Memories section
- User profile and achievement section
- Mobile-friendly navigation

## Tech Stack

- React
- JavaScript
- CSS
- Vite
- Lucide React icons
